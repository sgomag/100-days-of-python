# Day 23 – Turtle Crossing Game


A Frogger-style crossing game built using the Turtle library. The goal is to help a tortoise cross a busy road without getting hit by oncoming traffic.

This was a capstone project in the bootcamp, meaning I built it myself without step-by-step instructions. I also extended the project by adding a method to the CarManager class that prevents newly spawned cars from overlapping with recently created ones.
