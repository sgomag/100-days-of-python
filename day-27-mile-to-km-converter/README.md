# Day 27 – Mile to Km Converter

A simple GUI application that converts miles to kilometres. The user types a distance in miles, clicks Calculate, and the result appears on screen.

This was my first project building a GUI with Tkinter, learning the basic building blocks (labels, entry fields, and buttons). I also learned about `*args` and `**kwargs`, and used `**` to unpack a dictionary of padding values directly into widget constructors to avoid repetition.

`Other_Tkinter_Widgets.py` is a separate file with examples of other Tkinter widgets, kept as a reference.
