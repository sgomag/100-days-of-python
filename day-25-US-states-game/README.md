# Day 25 – US States Game
A geography quiz game built with Turtle and pandas. A blank map of the USA is displayed, and every time the user guesses a state correctly, its name appears in the right location on the map. By typing `exit`, the game ends and all missed states are shown in red.

This was my first project working with CSV files and pandas DataFrames. The core skill was filtering a DataFrame by row to retrieve the x and y coordinates of each guessed state, so it could be placed on the map.
